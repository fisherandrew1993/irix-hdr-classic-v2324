## Overview

Irix HDR Classic is built for HDR workflows that work best with a clear, visual editing process. It combines exposure merging, tone mapping, and preview-driven adjustments in one desktop application, so photographers can move from source frames to a finished HDR image with fewer steps in between.

The software focuses on day-to-day creative production: assembling bracketed photos, checking edits live as they are applied, and exporting in batches when the same treatment needs to be repeated across multiple files. With multilingual support and plugin compatibility, it can slot into a broader editing stack rather than acting as a standalone utility.

## Usage

A standard workflow is:

1. Import the bracketed photos or source images you want to process.
2. Merge the set into an HDR composition.
3. Adjust tone mapping while monitoring the real-time preview.
4. Apply ghost removal if moving subjects affect the result.
5. Export the final image, or queue several files for batch export.

For plugin-based workflows, load the relevant extension before starting the edit session so it can be available during processing.
